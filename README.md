# Customer Churn Prediction  
# Previsão de Churn de Clientes

---

## 🇺🇸 English Version

## 📌 Overview

This project presents a **Customer Churn Prediction model**, a classic Machine Learning problem widely used in industries such as **telecommunications, banking, and subscription-based services**.

The goal is to identify customers who are likely to stop using a service, enabling proactive retention strategies.

---

## 🧠 Business Problem

Customer churn directly impacts revenue and growth.

Being able to predict churn allows companies to:

- Reduce customer loss
- Improve retention campaigns
- Increase customer lifetime value (CLV)
- Optimize marketing strategies

---

## ⚙️ Approach

This project follows a standard ML pipeline:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training and evaluation
- Performance comparison

---

## 🧪 Methodology

1. Data exploration  
2. Feature preparation  
3. Model training (classification)  
4. Evaluation (accuracy, precision, recall, etc.)  
5. Analysis of results  

---

## 📊 Results

The model is able to:

- Identify patterns related to churn
- Highlight key features driving customer behavior
- Support decision-making in business context
